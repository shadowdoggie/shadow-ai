📋 First time connecting Google? Step-by-step guide

Google features are optional — Shadow AI works fully without them. You'll make your own free Google Cloud credentials (about 5–10 minutes, one time). Because the app is yours, no Google verification is needed.

  1. Create a Google Cloud project. Open project creator →
  2. Enable the APIs you want to use: Gmail, Calendar, Contacts (People), Drive.
  3. Configure the OAuth consent screen. Open consent screen → Choose External, fill the basic info, and under Test users add your own Google email — this is what lets it work without Google verification.
  4. Create an OAuth client. Open credentials → Pick Create credentials → OAuth client ID → Web application. Under Authorized redirect URIs, add this exact URL:
  5. When the client is created, Google shows its Client ID and Client secret — copy both and paste them into the Connect with Client ID + Secret fields below. If the secret is hidden (Google no longer lets you view it later), click + Add secret to make a fresh one and copy it immediately. (If you happened to get a JSON file, you can use Import Credentials JSON instead.)

⚠️ The first time you connect, Google shows a “Google hasn't verified this app” screen. That's normal for your own app — click Advanced → Go to Shadow AI (unsafe) to continue. Only you can use it unless you choose to publish & verify it yourself.
